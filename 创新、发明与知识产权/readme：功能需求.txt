功能需求：
对脚本所在目录下的以“姓名-学院-专业班级-学号-联系方式”格式命名的pdf文件，将其命名数据导出到Excel表格“正确统计”，以“-”为标志，将不同的信息放到不同列中，对没有按照这种格式命名的文件（pdf文件），以其文件名称中“院”字为标识，导出其名称到“院”字的部分到姓名栏，剩下的导出到第二栏，如果没有“院”字，只对其进行标记计数并导出名称到不正确表格。表格需要对导出的数据按照学院进行排序并去重，在处理完数据后，控制台输出本次处理的文件数量（按格式进行统计）和成功导出到表格的文件数量。
在导出正确命名的信息后，将正确命名的文件转移到该目录下的文件夹“正确”下，不正确的转移到“不正确”下（没有就创建一个），导出的数据表格也分别放到两个文件夹下。
